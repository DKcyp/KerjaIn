# Reports Module - Quick Start Guide

## 🚀 Getting Started

### Access the Reports Page
Navigate to: **`/reports`** in your browser

## 📊 Available Reports

### 1️⃣ Laporan Proyek (Project Reports)
**What it shows**: Distribution of projects across different stages

**How to use**:
1. Click the "Laporan Proyek" tab
2. View the horizontal bar chart showing project distribution
3. Scroll down to see detailed project table
4. Click column headers to sort

**Key Metrics**:
- Projects per stage (Blueprint, Development, UAT, EUT, Go-Live)
- Project progress percentages
- Client and PM information

---

### 2️⃣ Laporan Task (Task Reports)
**What it shows**: Task completion status and overdue tasks

**How to use**:
1. Click the "Laporan Task" tab
2. View metric cards at the top:
   - 🟢 Task Selesai (Completed)
   - 🔴 Task Overdue (Late)
   - 📊 Donut chart (Visual distribution)
3. Review task table below

**Key Metrics**:
- Total completed tasks
- Overdue tasks count
- Visual status distribution

---

### 3️⃣ Laporan UAT/EUT (Testing Reports)
**What it shows**: Quality assurance and bug tracking

**How to use**:
1. Click the "Laporan UAT/EUT" tab
2. View four metric cards:
   - Total Test Cases
   - Pass count (with %)
   - Fail count (with %)
   - Outstanding Bugs
3. Review bug table with severity levels

**Key Metrics**:
- Test case pass/fail rates
- Bug severity distribution
- Outstanding issues

---

### 4️⃣ Laporan Tiket Support (Support Reports)
**What it shows**: Customer support performance and SLA compliance

**How to use**:
1. Click the "Laporan Tiket Support" tab
2. View three metric cards:
   - Total Tickets
   - Resolved Tickets
   - SLA Achievement %
3. Review ticket table with resolution times

**Key Metrics**:
- Total tickets received
- Resolution rates
- SLA compliance percentage

---

## 🔍 Filtering Reports

### Date Range Filter
1. Click the "Pilih Rentang Waktu" input
2. Select start date
3. Select end date
4. Filter applies to all tabs automatically

### Project Filter
1. Click the "Semua Proyek" dropdown
2. Select specific project or keep "Semua Proyek"
3. Filter applies to all tabs automatically

---

## 💾 Exporting Reports

### Export to Excel
1. Click the **"Export"** button (top right)
2. Select **"Export ke Excel"**
3. File downloads automatically as `.xlsx`
4. Open in Microsoft Excel, Google Sheets, or LibreOffice

**What's included**:
- Current tab data
- All visible columns
- Formatted headers
- Auto-sized columns

### Export to PDF
1. Click the **"Export"** button

## 🎨 Understanding Color Codes

### Status Colors
- 🟢 **Green**: Success, Completed, Pass, SLA Met
- 🔴 **Red**: Overdue, Fail, Critical, SLA Missed
- 🔵 **Blue**: In Progress, Information, Total counts
- 🟠 **Orange**: Warning, Outstanding, Major issues
- 🟣 **Purple**: Special metrics (SLA percentage)
- 🟡 **Yellow**: Minor issues

### Project Stages
- 🟠 **Orange**: Blueprint
- 🟣 **Purple**: Development
- 🔵 **Blue**: UAT/EUT
- 🟢 **Green**: Go-Live

### Bug Severity
- 🔴 **Red**: Critical
- 🟠 **Orange**: Major
- 🟡 **Yellow**: Minor

## 💡 Tips & Tricks

### Quick Navigation
- Use tabs at the top to switch between report types
- All filters persist across tabs
- Click column headers to sort tables

### Best Practices
1. **Set date range first** for accurate reporting
2. **Filter by project** for focused analysis
3. **Export regularly** for historical tracking
4. **Check overdue tasks** in Task Reports daily
5. **Monitor SLA** in Support Reports weekly

### Keyboard Shortcuts
- `Tab`: Navigate between elements
- `Enter`: Activate buttons/links
- `Arrow Keys`: Navigate dropdowns

## 🔧 Troubleshooting

### No Data Showing?
- Check if date range is set correctly
- Verify project filter is not too restrictive
- Ensure you have permission to view the data

### Export Not Working?
- Check browser download settings
- Disable popup blockers
- Try a different browser

### Charts Not Displaying?
- Refresh the page
- Clear browser cache
- Check internet connection

## 🎯 Common Use Cases

### Daily Standup
1. Open **Laporan Task**
2. Check overdue tasks (red cards)
3. Review today's tasks

### Weekly Review
1. Set date range to last 7 days
2. Check all tabs for progress
3. Export to Excel for records

### Monthly Reports
1. Set date range to last month
2. Visit each tab
3. Export each report
4. Compile into presentation

### Client Meetings
1. Filter by specific project
2. Show **Laporan Proyek** for progress
3. Show **Laporan UAT/EUT** for quality
4. Export for client records
